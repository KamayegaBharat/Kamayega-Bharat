# Security Policy for Kamayega Bharat 🛡️

At Kamayega Bharat, the security of our users' data—especially their emails and authentication details—is our top priority. We maintain a 100/100 performance and security standard.

## Supported Versions

We currently only provide security updates for the latest version of the platform.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | ✅ Yes             |
| < 1.0   | ❌ No              |

## Reporting a Vulnerability

Please do not open a public GitHub Issue for security vulnerabilities.

If you discover a potential security risk or a bug that could expose user data, please report it privately. This allows us to fix the issue before it can be exploited.

### How to report:
1. Email: Send a detailed report to Kamayegabharat@gmail.com
2. Subject: SECURITY VULNERABILITY - Kamayega Bharat
3. Details: Include steps to reproduce the vulnerability and any potential impact.

We aim to acknowledge all security reports within 24-48 hours and will provide a timeline for a fix.
